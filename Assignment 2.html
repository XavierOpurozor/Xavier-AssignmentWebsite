CSS Selector Types (For Reference)

Here are the common types of CSS selectors. This page only uses element selectors for styling.

  • Element Selector – targets HTML tags (e.g., p, h1, table)
  • Class Selector – targets elements with a specific class (e.g., .highlight)
  • ID Selector – targets an element with a specific ID (e.g., #main-title)
  • Universal Selector – targets all elements (e.g., *)
  • Group Selector – styles multiple elements at once (e.g., h1, h2, p)
  • Descendant Selector – targets elements inside other elements (e.g., div p)
  • Child Selector – targets direct children (e.g., ul > li)
  • Attribute Selector – targets elements with specific attributes (e.g., input[type="text"])
  • Pseudo-class Selector – targets elements in a specific state (e.g., a:hover)
